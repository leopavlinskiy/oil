# Задача: Анализ рынка масел и разработка стратегии выхода на российский рынок

## Контекст

### Производитель
- **Завод**: Дубайский завод, производящий масла под брендами Pemco, Manol, Фаворит
- **Качество**: Масла международного качества с европейскими присадками (используются в маслах Shell, Mobil и др.)
- **Особенность**: Завод строго соблюдает санкционные ограничения, не занимается прямыми поставками в Россию

### Текущая ситуация
- Небольшие объемы поставляются через казахстанскую компанию (фура в неделю)
- Логистическая цепочка неэффективна (Дубай → Алматы → Россия)
- Казахстанская компания работает с российским рынком "по остаточному принципу"
- Есть один крупный покупатель в России: [tmtr.ru](https://www.tmtr.ru) (сервисная сеть)

### Преимущества
- Прайс-лист на 30% дешевле, чем у других покупателей завода
- Отсрочка оплаты от завода
- Прямая логистика Дубай → Москва (в 2 раза быстрее и дешевле)
- Казахстанская компания готова передать существующий объем

### Финансовые требования
- **Стоимость компании в Дубае**: примерно 150 тыс. долларов в год
- **Требование**: бизнес должен окупить как минимум эти затраты
- **Дополнительная цель**: через эту компанию планируется проводить другие товары (техника, запчасти)
- **Критический момент**: один контракт на фуру в месяц (~120-150 тыс. долларов) скорее всего не обеспечит нужной доходности для окупаемости компании
- **Вывод**: необходимы дополнительные клиенты в России для обеспечения рентабельности
- **Текущее состояние**: пока фура идет через Казахстан, затраты на компанию в Дубае не актуальны, поэтому текущая ситуация устраивает собственников бизнеса

## Основная задача

Подготовить комплексную аналитику российского рынка масел и технических жидкостей и разработать стратегию создания компании-дистрибьютора для поставки продукции на российский рынок.

### 4. Стратегия выхода на рынок

#### 4.1. Краткосрочная стратегия (начальный этап)
- Работа с существующим покупателем (tmtr.ru)
- **Критически важно**: поиск дополнительных крупных оптовых покупателей
- Прямые поставки без промежуточных звеньев
- Объем: фура в неделю (~120-150 тыс. долларов)
- **Цель**: обеспечить объемы, достаточные для окупаемости компании в Дубае (минимум 150 тыс. долларов в год)

#### 4.2. Среднесрочная стратегия
- Развитие сети партнеров-дистрибьюторов
- Создание складской сети
- Работа с крупными оптовиками, у которых уже есть дистрибуционная сеть
- Совместная разработка планов действий с партнерами

#### 4.3. Долгосрочная стратегия
- Создание полноценной компании-дистрибьютора
- Возможное создание отдельной компании-импортера
- Розничные каналы (как следующий этап, требует больших объемов)
- Маркетплейсы
- Рекламная активность
- **Поставка присадок российским заводам**:
  - Отдельное направление бизнеса с высокой добавленной стоимостью
  - Российские заводы в основном используют китайские присадки
  - Возможность поставки качественных европейских присадок
  - Создание регулярных поставок и встраивание в производственный цикл заводов
  - Более узкая линейка, но с высоким потенциалом
- **Разлив масла в России**:
  - Возможность закупки масла кубом в Эмиратах
  - Импорт в Россию в крупной таре (IBC, бочки)
  - Организация разлива по бутылкам в России
  - Разлив под собственным брендом или существующим брендом (Pemco, Manol, Фаворит)
  - Требования к производству для разлива (лицензии, сертификация, оборудование)
  - Экономическая целесообразность разлива в России vs. импорт готовой продукции
  - Партнерство с компаниями, которые уже занимаются переразливом чужого масла под своим брендом
  - Возможности совместного использования производственных мощностей
- Использование компании в Дубае для других товаров (техника, запчасти) после обеспечения рентабельности по маслам

### 5. Дополнительные направления

#### 5.1. Брендинг
- Текущий бренд: Pemco
- Возможность использования других брендов завода (Manol, Фаворит)
- Возможность создания собственного бренда
- Брендирование продукции

#### 5.2. Ассортимент
- Полная линейка продуктов (см. файл с прайс-листом)
- Разные фасовки: 1л, 4л, 5л, 20л, бочки (208л), IBC (1000л)
- Разница в цене по фасовкам (примерно 12% между 20л и кубом)

**Основные направления продукции:**
- **Group I Automotive Oils** — автомобильные масла премиум-класса
- **Group II Automotive Oils** — автомобильные масла стандартного класса
- **Group III Industrial** — промышленные масла (гидравлические, редукторные, компрессорные, турбинные, для направляющих и др.)
- **Group IV Heavy Duty UHPD** — масла для тяжелой техники (UHPD — Ultra High Performance Diesel)
- **Group V Heavy Duty SHPD** — масла для тяжелой техники (SHPD — Super High Performance Diesel)
- **Operating & Service Fluids** — эксплуатационные жидкости (AdBlue и др.)
- **Group VI Marine Oils** — морские масла
- **Group VII Motorcycle oils** — мотоциклетные масла
- **Group VIII ATF** — трансмиссионные жидкости (ATF)
- **Group IX Gear Oils** — масла для коробок передач
- **Group X LMOA Oils** — масла LMOA

**Сертификации и стандарты (Features):**
Features — это колонка в каталоге/прайс-листе продукции, где для каждого продукта указаны его характеристики, сертификации, стандарты и допуски. Эти данные определяют качество и область применения масел. В описании Features для каждого продукта указаны:
- **SAE** — классификация вязкости (например, 5W-30, 10W-40)
- **API** — стандарты Американского института нефти (SN, SP, CH-4, CI-4, CK-4, CJ-4 и др.)
- **ACEA** — европейские стандарты качества масел (A3/B4, C2/C3, C4, C5, E7, E8, E9, E11 и др.)
- **ILSAC** — международные стандарты (GF-6A и др.)
- **Спецификации производителей техники:**
  - VW (Volkswagen) — например, VW 505 01
  - Volvo VDS-5
  - MB (Mercedes-Benz) 228.61
  - MAN M3977
  - Scania LDF-5
  - Cummins 20074
- **ISO** — международные стандарты для промышленных масел (ISO 32, 46, 68, 100, 150, 220, 320, 460, 680)
- **DIN** — немецкие стандарты (DIN 51515-1,2, DIN 51524-1, DIN 70070)
- **JASO** — японские стандарты (JASO FB, JASO FC, JASO MA/MA2, JASO M315 Type 1A)
- **TBN** — щелочное число (Total Base Number) для некоторых масел
- **Другие спецификации**: NMMA TC-W3, HLP, VDL и др.

#### 5.3. Поставка присадок
- **Возможность поставки присадок российским заводам**:
  - Российские заводы в основном работают с китайскими присадками
  - Завод в Дубае использует европейские присадки высокого качества
  - Поставка присадок может быть отдельным направлением бизнеса
  - Возможность создания регулярных поставок для российских заводов
  - Встраивание в производственный цикл заводов в качестве поставщика присадок
  - Присадки — ключевой компонент качества масел, российская база часто имеет проблемы со стабильностью качества
  - Поставка присадок может позволить российским заводам производить масла с использованием качественных европейских присадок
  - Это может быть более узкая линейка, но с высокой добавленной стоимостью

#### 5.5. Дополнительные возможности и партнерства
- **Партнерство с компаниями, занимающимися переразливом масел**:
  - Поиск компаний, которые переразливают чужое масло под своим брендом

  - Партнерство с производителями упаковки (бутылки, канистры)

  - Партнерство с логистическими компаниями для оптимизации доставки
